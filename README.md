# Salon

A Place To Get Fancy

## Setup Instructions

Requires PostgreSQL
```
Database Name = hair_salon

Table 1 = stylists
Table 2 = clients

Table 1 Columns = stylist_name
Table 2 Columns = client_name, stylist_id, client_phone
```

### Gems used
```
sinatra
sinatra-contrib
PG
```
### Install Bundler
```
$ gem install bundler
```
### Run Bundler
```
$ bundle
```
